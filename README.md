# AURA Android Wrapper

App Android nativa que envuelve AURA ∞.Ω en un WebView con permisos completos de micrófono y audio.

## URL
`https://jrrubimar7.github.io/aura-/`

## Compilar

GitHub Actions compila automáticamente el APK en cada push.

1. Haz push al repo
2. Ve a **Actions** → **Build AURA APK**
3. Descarga el artifact **AURA-debug-apk**

## Permisos
- Internet
- Micrófono (para CONV/VAD)
- Audio

## minSdk: 26 (Android 8+)
